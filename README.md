#  Tic Tac Toe Game (React)

A simple and interactive Tic Tac Toe game built using React.js with clean UI and basic game logic.

##  Features
- 🎯 2-player game (X vs O)
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Restart game option
- ⚡ Fast and responsive UI

##  Tech Stack
React.js, JavaScript, HTML, CSS

## Setup Instructions
git clone https://github.com/akhil5005/tic-tac-toe-Game.git  
cd tic-tac-toe-Game  
npm install  
npm start  

App runs at: https://tic-tac-toe-game-gray-alpha.vercel.app/

##  Build
npm run build
